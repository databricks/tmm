# Instructor Guide

STOP reading here as a student  

* You can run this workshop with Databricks Express, in Vocareum (see `config.json` file for Vocareum in this folder) or any other workspace other than Databricks Free Edition.
* As an instructor, use another browser that does not share cookies with your student browser, log in as **workspace admin** and run the loan broker notebook from this folder
* Running the loan broker notebook will set up the UC catalog, schema and permissions and then generate a stream of loan request  
* It will add a customer.json file to the volume for the Genie Code lab
* After running this setup notebook, make sure UC catalogs exist and the loan broker is running. The broker will continue to run for the length of the lab.
